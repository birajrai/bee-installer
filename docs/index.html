Bee Installer

Automatically install Nginx Proxy Manager (NPM) on an Ubuntu host using Docker.

Quick install

Run these on your server as root (or via sudo).

curl -fsSL https://raw.githubusercontent.com/birajrai/bee-installer/main/scripts/installers/installer-npm.sh | sudo bash
curl -fsSL https://raw.githubusercontent.com/birajrai/bee-installer/main/scripts/installers/installer-traefik.sh | sudo bash
curl -fsSL https://raw.githubusercontent.com/birajrai/bee-installer/main/scripts/installers/installer-postgres.sh | sudo bash
curl -fsSL https://raw.githubusercontent.com/birajrai/bee-installer/main/scripts/installers/installer-certbot.sh | sudo bash
curl -fsSL https://raw.githubusercontent.com/birajrai/bee-installer/main/scripts/installers/installer-nodejs.sh | sudo bash

After installation the NPM web UI will be available on port 81. See the project README for details.
